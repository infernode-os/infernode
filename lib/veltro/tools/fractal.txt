fractal - Show and control the Mandelbrot/Julia fractal viewer.
USE THIS TOOL whenever the user asks to see, render, or explore any
fractal (Mandelbrot, Julia, etc.). Do NOT try to render fractals as
mermaid or other diagrams — this tool drives a real fractal renderer.
This tool takes one string argument named "args". The first word of
"args" is the subcommand: state, view, zoomin, center, zoomout, julia,
mandelbrot, depth, fill, restart.
Workflow when the user asks for a fractal: first call the launch tool
with args = "fractals" to start the viewer, then call this tool.
Examples on this tool:
  args = "julia -0.4 0.6"                (Julia set for c=-0.4+0.6i)
  args = "mandelbrot"                    (switch to Mandelbrot set)
  args = "center -0.75 0.1 0.05"         (zoom on seahorse valley)
  args = "view"                          (read current view description)
Commands are processed once per tick (~500ms); send one at a time and
read state before sending the next.

Usage:
  state                          Read fractal state (type, coords, depth)
  view                           Read AI-friendly view description
  zoomin <x1> <y1> <x2> <y2>    Zoom to fractal coordinate region
  center <re> <im> <radius>     Zoom centered on a point with radius
  zoomout                        Go back one zoom level
  julia <re> <im>               Show Julia set for c = re + im*i
  mandelbrot                     Switch back to Mandelbrot set
  depth <n>                      Set depth multiplier (1=default)
  fill on|off                    Toggle boundary-trace fill mode
  restart                        Restart current computation

The fractal viewer must be running. Use 'launch fractals' to start it.

Commands are processed once per tick (~500ms). Send one command at a
time and wait for a state change before sending the next.

Coordinates are in the complex plane (clamped to ±4):
  Full Mandelbrot set: x=[-2, 1] y=[-1.5, 1.5]

Notable regions to explore:
  Seahorse valley:  center -0.75 0.1 0.05
  Elephant valley:  center 0.28 0.008 0.01
  Mini-Mandelbrot:  center -1.768 0.002 0.01
  Antenna tip:      center -1.788 0 0.02
  Period-2 bulb:    center -1.0 0 0.3

Examples:
  fractal view                            See current view description
  fractal state                           Read raw state
  fractal zoomin -0.8 0.05 -0.7 0.15      Zoom into seahorse valley
  fractal center -0.75 0.1 0.02           Zoom centered on point (aspect-corrected)
  fractal julia -0.4 0.6                  Show Julia set for c=-0.4+0.6i
  fractal depth 3                          Triple the iteration depth
  fractal zoomout                          Go back one level
  fractal mandelbrot                       Return to Mandelbrot set
