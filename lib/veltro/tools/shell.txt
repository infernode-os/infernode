shell - Read-only access to the shell terminal transcript.
Does NOT execute commands and does NOT start a shell. To start a
shell window, use the launch tool with args = "shell" — NOT this tool.
This tool takes one string argument named "args". The first word of
"args" is the subcommand: read, tail, status.
Examples on this tool:
  args = "read"          (read the full transcript)
  args = "tail 50"       (last 50 lines)
  args = "status"        (check if a shell is running)
If the user says "launch the shell" or "open a shell", use the launch
tool with args = "shell", NOT this tool.

Usage:
  read [body]     Read the shell transcript
  read input      Read the current input line
  tail [n]        Read the last n lines of transcript (default 30)
  status          Check if shell is running

NOTE: This tool is READ-ONLY for safety.
You cannot send commands to the shell.
Instead, propose commands in Chat for the user to run.

The shell must be running. Use 'launch shell' to start it.

Examples:
  shell read              Read the full transcript
  shell tail              Last 30 lines
  shell tail 50           Last 50 lines
  shell read input        Current input line
  shell status            Check shell status
