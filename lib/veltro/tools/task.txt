task - Create and manage delegated AI tasks

  task create label=<name> [tools=<csv>] [paths=<csv>] [urgency=<0-2>]
              [brief=<text>] [instructions=<text>] [category=<text>]
              [model=<name>] [agenttype=<type>]
      Create new task with isolated conversation and tools.
      Tools validated against delegation budget (/tool/budget).
      brief= sets the initial context message for the task agent.
      If omitted, a default prompt is generated from the label.
      instructions= sets structured directives injected into the
      TA system prompt via <instructions> tags. Use for specific
      procedural guidance the task agent should follow.
      model= selects the LLM backing the new activity. Valid:
      gpt-oss, daedalus, haiku, sonnet, opus. Omit to inherit the
      parent session's model.
      agenttype= picks a preset prompt + defaults. Currently:
        coder — Limbo-capable Daedalus fine-tune with
                tools=read,write,edit,find,grep,list,limbo
      Unknown args (e.g. priority=high) are rejected with the list
      of valid keys instead of being silently dropped.
      Returns activity ID.

  task status <id>     Show task status, urgency, label
  task list            List all active tasks with status
  task close <id>      Archive a completed task

Each task gets its own activity tile in the header bar, its own
conversation, tools, and filesystem overlay. Use for work that
should happen in parallel or needs security isolation.

Every child task automatically gets basic tools (read, list, find,
memory, todo, plan, present, gap). Use tools= to add task-specific
tools from the budget. Only budget tools may be delegated.

Use "task list" to monitor progress. Close tasks when done.

Examples:
  task create label=Review tools=grep,search
  task create label=Draft tools=write,edit urgency=1
  task create label=Research tools=grep,search,websearch
  task create label="Poetry" brief="Help the user write and refine poems"
  task create label=Editor instructions="Open /lib/veltro/system.txt and suggest improvements"
  task create label=Refactor tools=read,edit,grep instructions="Refactor all uses of foo() to bar()"
  task create label=BugFix agenttype=coder brief="fix the bug in cat.b"
  task create label=Port model=daedalus tools=read,write,edit,limbo brief="port spawn.b to ARM64"
  task list
  task status 2
  task close 2
